# Smart Home Control – UI/UX Design

Smart Home Control is a modern and minimal UI/UX design for managing home appliances in one app. The interface focuses on simplicity, real-time control, and a smooth user experience, allowing users to monitor and adjust devices like lights, fans, and thermostats easily. Designed in Figma with a focus on accessibility and modern aesthetics.

# Features
- Control multiple home appliances in one place  
- Real-time device monitoring  
- Clean and intuitive navigation  
- Modern visual design with a light aesthetic

##  Design Process
1. **Research** – Understanding user needs for smart home management  
2. **Wireframing** – Basic structure and layout planning  
3. **UI Design** – Created clean and consistent visual components  
4. **Prototyping** – Linked screens to simulate user flow in Figma

## Tools Used
- **Figma** – UI Design & Prototyping  
- **Canva** – Presentation & Collage (optional)
